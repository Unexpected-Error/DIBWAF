# ShatteredReality

Developed with Unreal Engine 4

## Features

* Main menu
	* Server Browser/creator

* HUD
	* Health bar

* Pause menu
  * Game start
  * Disconnect

* Damage system
	* Take damage
	* Spectator



* Win/Loss
	* Last man alive wins, match restarts after victory

* Advanced movemnt
  * B-hopping
  * Wall running
  * Air strafing
  * Edge/End boosting

* Portals
  * two per person


## Bugs
